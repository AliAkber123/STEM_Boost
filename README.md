# STEM_Boost
STEMBoost, an educational platform specifically engineered to assist low-vision learners in studying technical subjects without barriers. The system prioritizes accessibility through audio instructions, screen reading capabilities, and keyboard-only navigation to eliminate the need for a mouse pointer.
